Javascript

JavaScript é uma linguagem de programação voltada para o desenvolvimento web, permitindo criar interatividade e funcionalidades dinâmicas nas páginas.

Python

Python é uma linguagem amplamente usada em diversas áreas, como desenvolvimento web, automação, ciência de dados, inteligência artificial e mais, com suporte para várias bibliotecas e frameworks.

SQL

SQL é uma linguagem usada para gerenciar e manipular bancos de dados relacionais. Com ela, é possível consultar, inserir, atualizar e deletar dados, além de criar e modificar a estrutura de tabelas.
